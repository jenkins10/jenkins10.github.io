Missions & Experience

Specialized in autonomous onboard navigation systems — from concept through flight. I bridge the gap between GN&C algorithms and production flight software.

NASA DART Mission

Navigation engineer for SMART Nav — the autonomous system that guided the spacecraft to impact asteroid Dimorphos in September 2022. Designed the optical targeting algorithm to discriminate Dimorphos from Didymos in real-time imagery.

Optical Navigation • Autonomous Targeting

NASA Dragonfly Mission

Electro-Optical Terrain Sensing Function Lead for the Mobility subsystem. Led a team through conceptual development, analysis, implementation, and testing of onboard optical navigation for a rotorcraft exploring Titan.

Image Processing • Team Lead • Flight Software

Autonomous Station Keeping

Principal Investigator designing onboard guidance and navigation algorithms for station-keeping in chaotic dynamical environments. Architected a modular software framework for mission concept analysis.

GN&C • PI • Software Architecture
